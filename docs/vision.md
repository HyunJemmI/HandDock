# Vision

HandDock is a single local web entry point that opens many creative works.

Principles:

- One portal, many linked projects.
- Gesture-first interaction before mouse-first interaction.
- Camera plus hand relationship should drive navigation and control.
- Community code may be imported, but must be adapted into reusable local packages.

Near-term build order:

1. Build the hub shell.
2. Build a hand-tracking lab for gesture experiments.
3. Define a shared gesture API.
4. Integrate and adapt Spline community examples.
5. Connect each finished work back into the hub.
