# HandDock

HandDock is a local portal browser for gesture-driven projects.

The core idea:

- `apps/hub`: the local browser-like portal that links into each work.
- `apps/hand-tracking-lab`: the first playground for camera and hand interaction.
- `packages/hand-tracking`: shared tracking logic and gesture mapping.
- `packages/spline-adapters`: imported and modified Spline community code.
- `packages/ui`: shared portal UI components.

This repository is intentionally lightweight at the start. It is meant to support fast iteration, imported experiments, and later consolidation.
